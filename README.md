# Predictive Modeling of Electric Vehicle Battery Degradation

## Project Overview
This project applies Machine Learning techniques to predict the State of Health (SoH) and remaining useful life of Electric Vehicle batteries, building upon foundational data science comparisons of Nickel Manganese Cobalt (NMC) and Lithium Iron Phosphate (LFP) chemistries. 

## Repository Structure
* `data/`: Contains the synthetic dataset (`ev_battery_degradation_v1.csv`).
* `ev-battery-degradation-modeling.ipynb`: The primary Jupyter Notebook containing all mathematical formulations, Python code, and analysis.
* `README.md`: Project documentation and setup instructions.
